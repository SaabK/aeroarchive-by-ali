# Archive by Ali

A curated collection of aerospace engineering resources, software, courses, books, research databases, and learning tools.

## Features

* Organized engineering resource directory
* Fast search functionality
* Resource bookmarking
* Featured resources section
* Senior drive collection
* Engineering links directory
* Dark mode support
* Mobile-friendly design
* Lightweight and fast

## Purpose

Engineering students spend years collecting useful tools, software, websites, books, and courses.

Archive by Ali brings them together in one organized place.
